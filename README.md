# UX101 — Учимся проектировать UX

Образовательный блог-дневник, посвящённый погружению в профессию UX-проектировщика.

## О проекте

Сайт представляет собой серию из 21 главы, охватывающих путь от основ UX до роли продукт-менеджера. Материалы основаны на вебинарах, книгах и практических упражнениях.

Главы оформлены в виде статического сайта на генераторе Hexo. Но можно просто читать главы в `source\_posts\`, не устанавливая генератор.

## Технологии

- **Генератор:** [Hexo](https://hexo.io/) 3.2
- **Тема:** Timly (кастомная тема на базе Pug + Sass)
- **Язык:** русский

## Установка и запуск

```bash
npm install
hexo server
```

Сайт будет доступен по адресу `http://localhost:4000`.

## Сборка для публикации

```bash
hexo generate
```

Статические файлы будут сгенерированы в директорию `public/`.
